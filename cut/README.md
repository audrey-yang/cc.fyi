# `cut`

Following the [Build Your Own cut Tool](https://codingchallenges.fyi/challenges/challenge-cut) challenge.

## How to run

Compile the executable with `make` and run `./cut`. Refer to `cut` for usage.

## Comparison to `cut` and the Coding Challenge specs

Unlike the specifications from the website, I have chosen to implement LIST as described in the `cut` man pages. This implementation does not currently support cutting by byte or by character.
